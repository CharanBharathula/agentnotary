# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its Code of Conduct. By participating in this project (issues, PRs, discussions, etc.) you agree to abide by it.

## Reporting

Concerns can be raised privately by opening a GitHub issue with the `conduct` label, or by emailing the maintainer listed in `pyproject.toml`. Reports are handled confidentially.

## Scope

Applies to all project spaces — repository, issues, pull requests, discussions, and any official communication channel.
